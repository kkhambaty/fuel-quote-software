# fuel-quote-software

To run web app:
  Go to the fuel quote web app directory (after cloning the url) and type 'flask run' to start the app.
  Copy the url and add '/logi' to the end and enter. This should load the login page. the fuelQuoteAppData database is also necessary. The exported .sql file is in the repository which can be used to add the database locally.
